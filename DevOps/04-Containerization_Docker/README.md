### **02-Dockerfile_And_Images/** 📦
**Time Investment:** 1 week
- Writing effective Dockerfiles

### **03-Container_Networking_Storage/** 🌐
**Time Investment:** 1 week
- Docker networking models

- Data persistence strategies

### **04-Docker_Compose/** 🔧
**Time Investment:** 1 week

- Docker Hub and ECR usage
